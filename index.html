📁 Directory Commands

ls — List directory contents.

ls -a — List all files including hidden ones.

pwd — Show current directory.

cd / — Go to root directory.

mkdir folder — Create a directory.

📄 File Commands

nano file.txt — Edit file using nano editor.

cat file.txt — Display file contents.

🔐 Permissions Commands

chmod 755 filename — Change file permissions.

⚙️ System & Utility Commands

uname -a — System info.

🔁 Redirection Commands

echo Hello > file.txt — Redirect output to file.

📦 YUM & GCC Commands

yum install packagename — Install package using YUM.

gcc hello.c -o hello — Compile C program.

🧠 System Calls

fork() — Create a new child process.

💡 Shell Commands

test -f filename — Check if file exists.
